# GanttRO - Logiciel d'Ordonnancement (React + Vite)

## 1. Project Overview

- **Project name**: GanttRO
- **Type**: Web application (React + Vite)
- **Core functionality**: Gantt chart scheduling tool with step-by-step visualization
- **Target users**: Project managers, students, professionals

## 2. UI/UX Specification

### Color Palette (Light theme)
- Primary: #2196F3 (Blue)
- Secondary: #4CAF50 (Green)
- Accent: #FF9800 (Orange)
- Background: #FAFAFA
- Surface: #FFFFFF
- Text Primary: #212121
- Text Secondary: #757575

### Components
1. **Task Input Table**: Task (A-Z), Duration, Predecessors
2. **Navigation Controls**: Previous/Next buttons with step indicator
3. **Gantt Chart**: Horizontal bars showing task execution

## 3. Acceptance Criteria
- [ ] User can add/remove tasks
- [ ] Schedule calculated based on dependencies
- [ ] Step-by-step visualization works
- [ ] Light color theme used throughout
